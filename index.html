Oracle Helpers

Using the helper below will let you run the following functions:

  1. Everywhere

    • Show a tooltip for any highlighted number to convert it from a decimal number (e.g. 8.5) into days, hours and minutes (e.g. 1 day, 8h30m)
    • Tries to find any decimal date and automatically shows a convertion to hours and minutes

  2. Web Clock Page

    • Show the total clocked-in time in addition to the "Elasped Time" at the top of the page. E.g. at the end of the day it would show 7 hours rather than just the amount of time since you clocked in after lunch.

  3. Time Card Page

    • The the full decimal hours under reported time, rather than a rounded up number. It will also conver this to days, hours and minutes.
    • Show the total time for each day

  4. Existing Absences Page

    • Shows a calendar to show all absences for the year

Please note that currently, all "day" calculations assume a 7 hour working day.

How to install

To install the Oracle Helpers, drag and drop the "Oracle Helpers" link below to your favorites bar.

How to use

To use the oracle helpers, navigate to one of the pages above and click on the "Oracle Helpers" link in your favorites. The new content will be displayed on the screen where appropraite.

The "tooltip" that converts decimal numbers will show whenever you hightlight any number.

Please note: while this code does not transmit any information to any exteral services and all processing is done locally in your browser, you should always be careful when using code shared online.

The full source code for this tool can be found at https://github.com/RoyMudie/OracleHelpers.
